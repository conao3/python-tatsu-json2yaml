# python-tatsu-yaml

YAML parser powered by TatSu.

# Usage

Install `tatsuyaml` in local Python.

```sh
python -m venv .venv
. .venv/bin/activate
pip install -e .[dev]
```

Then, you could run `tatsuyaml`.

```sh
$(.venv) tatsuyaml
cli.py: hello
```

And you want to leave this virtualenv, execute below command.

```sh
$ deactivate
```
